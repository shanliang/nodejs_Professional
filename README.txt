Professional Node.js - Building Javascript Based Scalable Software
------------------------------------------------------------------

Code samples:
-------------

This folder includes the code samples used in the numbered code listings for the Professional Node.js (Wrox) book available here:

http://www.wrox.com/WileyCDA/WroxTitle/Professional-Node-js-Building-Javascript-Based-Scalable-Software.productCd-1118185463.html

Chapters 1 to 7 and 9 don't have code listings.


Usage:
------

Each file is named sequentially, but for them to work you should follow the instructions on the book and rename the files as specified there.

For this code to work you also need to have Node.js installed as described in the first chapters of the book.
